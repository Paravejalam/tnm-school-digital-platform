# AI Prompts Index

## 1. Purpose

This folder contains the AI prompt assets and guidance used to support documentation, analysis, and implementation workflows for the T.N. Memorial Public School Digital Platform.

## 2. Folder Purpose

The AI prompts area is organized to support repository-aware assistance across documentation, architecture, testing, deployment, and operational tasks.

## 3. Prompt Organization

The following subfolders are available:

- ChatGPT: prompts tailored for ChatGPT-assisted workflows.
- DeepSeek: prompts tailored for DeepSeek-assisted workflows.
- Qwen: prompts tailored for Qwen-assisted workflows.
- Prompt_Templates: reusable prompt structures for common documentation and engineering tasks.

## 4. AI Workflow

The repository’s AI workflow should:

1. Reference repository documentation and naming conventions.
2. Preserve the existing documentation structure.
3. Produce implementation-oriented and repository-consistent outputs.
4. Avoid unsupported assumptions or new technology choices.

## 5. Best Practices

- Use prompts that are specific and scoped to the requested task.
- Reference existing documentation before generating new content.
- Validate AI-generated outputs against repository standards.
- Keep prompts aligned with the project’s architecture, security, and delivery model.
