# T.N. Memorial Public School Digital Platform

## Project Overview

The T.N. Memorial Public School Digital Platform is an enterprise-grade school website and ERP platform designed to support modern school operations through a professional web experience, structured administration workflows, and comprehensive technical documentation.

The repository now includes complete documentation coverage across architecture, API, database, UI, security, testing, deployment, portfolio, ADR, RTM, QMS, appendix, and AI prompt guidance.
